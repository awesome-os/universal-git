# @awesome-os/isomorphic-git
Is a isomorphic-git fork that is a fascade for @awesome-os/universal-git universal-git and is mainly there for Projects that depend already on legacy isomorphic-git and want to have a easy Migration Path.

But it also bundels the efforts to polyfill backward compat and makes universal-git run able on near any platform even none WInterCG Compatible
EcmaScript environments like legacy React nativ pre 0.74 without stream support.

This implements Stream Support and so on for legacy Platforms.
